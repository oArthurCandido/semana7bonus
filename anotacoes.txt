O que é o NEXTJS?

É um framework baseado em React. Ele renderiza o react no lado servidor. Em aplicações React tradicionais toda a interface e chamadas à API é feita pelo lado browser, o que leva a uma espera demasiada por parte do usuário.

SSR = Server Side Rendering

Com isso conseguimos aplicações React mais performáticas e melhor indexadas no google.

O que é um Framework Web?

É um sistema opinativo com estrutura e feraamentas já definidas. Eles ajudam no desenvolvimento rápido e seguro de aplicações.
